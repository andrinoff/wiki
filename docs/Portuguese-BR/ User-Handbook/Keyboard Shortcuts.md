# Atalhos de Teclado Essenciais

| Atalho | Ação |
|----------|--------|
| Super (Windows key) | Abre Menu de Aplicativos |
| Super (Windows key) + A | Mostra Tela de Aplicativos|
| Super (Windows key) + L | Bloqueia a tela |
| Alt + F4 | Fecha a Janela |
| Print Screen | Tira uma Captura de Tela |

# Áreas de Trabalho e Multitarefa

GNOME usa áreas de trabalho dinâmicas.

![Visão geral da Área de Trabalho](/img/workspaces-overview.png)

## Criando uma Área de Trabalho
Arraste uma janela para a pré-visualização da área de trabalho vazia ou coloque um número customizado de áreas de trabalho nas configurações.

![Criação de Área de Trabalho](/img/workspace-creation.png)

- Super + Tab

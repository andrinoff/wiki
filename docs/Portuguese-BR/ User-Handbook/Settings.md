# Configurações do Sistema

Abra **Configurações** no menu do sistema.

![Visão geral de Configurações](/img/settings-overview.png)

## Rede

Configure conexões Wi-Fi e Ethernet em **Rede**.

![Painel de Rede](/img/network-panel.png)

## Tela

Ajuste resolução, escala e layout em **Telas**.

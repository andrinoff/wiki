# Instalando Aplicativos no Oreon

Oreon usa GNOME Software para instalar aplicativos.

![Centro de GNOME SOFTWARE em Oreon](/img/softwarecenter.png)

## Usando GNOME Software

1. Abra **Software** na visão geral de Atividades.
2. Procure pelo aplicativo.
3. Clique **Instalar**.

![Centro de GNOME SOFTWARE em Oreon](/img/softwarecenter1.png)

## Usando o terminal

```bash
sudo dnf install <package-name>

# Customizando a Área de trabalho

## Alterando o Papel de Parede

1. Abra **Configurações**
2. Selecione **Aparência**
3. Escolha um papel de parede

![Seletor de papel de parede](/img/wallpaper-selector.png)

## Modo Escuro e Modo Claro

Ative ou desative o modo escuro na central de controle.

![Ativar/desativar modo escuro](/img/dark-mode-toggle.png)
